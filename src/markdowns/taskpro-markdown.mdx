
#  TaskPro 📚

🚀 Built with **Next.js** 🚀

TaskPro is a comprehensive full-stack application designed to streamline task management. It empowers you to effortlessly organize tasks by priority, status, and category.

## Features 🎉

- **Task Creation**: Easily create tasks and add descriptions to keep track of your work. 📌
- **Task Category**: Organize tasks more effectively by grouping them into categories. 📝
- **Task Prioritization**: Prioritize your tasks with customizable labels and tags for better organization. 🔄
- **Board with Status**: Organize tasks on a board with **To Do**, **In Progress**, and **Completed** status columns. 📋
-  **Text Editor:** Edit task descriptions and notes with a text editor. ✏️
-  **Google Authentication:** Securely log in to TaskPro using your Google account for easy access. 🔑
